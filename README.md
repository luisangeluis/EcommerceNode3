## CARTITEM Routes
- /api/v1/products/:id/add-to-cart
- - POST "Add product to cart as customer"

- /api/v1/cartItem/:cartItemId
- - GET "Get a cart item as customer"
- - PATCH "Update quantity in cart item"
- - DELETE "Remo a cart item"
